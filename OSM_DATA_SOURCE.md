# OpenStreetMap Data Source

## Cebu Municipality Boundaries

**Branch:** `openstreetmap`

**Data File:** `public/cebu-municipalities.geojson`

**Source:** OpenStreetMap-derived administrative boundaries

**Coverage:** 54 municipalities and cities in Cebu Province, Philippines

- All municipality boundaries with accurate island and coastal geometry
- Includes Cebu City and its surrounding municipalities
- Proper polygon geometry for all administrative divisions
- MultiPolygon structures preserved for municipalities with multiple island components

**Last Updated:** April 10, 2026

**Format:** GeoJSON FeatureCollection

**Features per municipality:**
- name: Municipality/City name
- type: Classification (Municipality/City)
- province: "Cebu"
- region: "Central Visayas (Region VII)"
